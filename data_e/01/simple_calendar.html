Simple Calender

Simple Calender

Confirmed operation:
MacOS 10.14.6 / Xcode 11.3.1
Display dates of one month. Starts Monday and ends on Sunday. A number of weeks will be 5 or 6 weeks. The beginning of a month is filled with the last days of the previous month, and the end of the month is filled with the first days of the following month.
Click month move button (< or >) to display the calendar for the next month or the previous month. Push arrow keys on your keyboard to move dates.
Saturdays, Sundays and public holidays change the color of the date. Holidays are managed based on the holiday file which will be made by you .

View hierarchy

A date is displayed in a view object created from the subclass of NSView class which customizes appearance of the date. Objects which display dates are placed in a calendar view which is parent view.

Class list

AppDelegate
Create an object of UAView class and add it on content view.
UAView：NSViewのサブクラス
Display a calendar for one month. Holds a set of objects of UAItemView class that displays a date. UAView class and UAItemView view class have a parent-child relationship.
UAItemView：NSViewのサブクラス
Display a day string and have display-related attributes (font, color, etc.)
UACalendar
Holds a set of UACalendarDate objects needed to display a calendar of a month. UACalendar class and UACalendarDate class have a parent-child relationship.
UACalendarDate
Display a date. Have display-related attributes (font, text color, etc. )
Looking at the above classes in MVC model, UAView class and UAItemView class correspond to model layer, and UACalendar class and UACalendarDate class correspond to view layer.
UADateUtil
A singleton class that provides service methods for date operations. Getting a NSDate object for the first or last day of a particular month, getting the number of days in a particular monthm, addition and subtraction of dates,Japanese calendar conversion, and so on.
UATextField
A subclass of NSTextField that display headings.
NSColor+MyColor.h
A category of NSColor class. Create a color object of CGColor with ​RGB
holiday.json
The file that registered calendar holidays. It is a JSON-formatted file that you put in your application bundle resources. The key is a date and the value is a holiday name.
