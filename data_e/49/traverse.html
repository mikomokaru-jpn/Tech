Full-text search tool

Full-text search while traversing directories

Search for all text files under a particular directory, including subdirectories, Output a list of files containing character strings that matches the keyword.
Click the image to see the video.

How to operate

Click 選択 Button to open OpenPanel and select a directory from which you want to start your search.
Enter a search word to search. Only one search word can be specified. It is not case sensitive. A full-text search has been performed , as a result, if there is matching files, the list of folder name, file name, number of hits, and file size are displayed in the table view.
Double-click a file name of Table View to open the file with the text editor (mi).

Search history log output

A serach result that consist a search date, a search word, and the number of hit files is saved as a history record in UserDefaults.
You can output this data to a file. Select menu → History → Output history, and Save Panel will be displayed, so set a output destination file. After "Save" has been executed, the search history data is output to the file.
