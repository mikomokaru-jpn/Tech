Open an image file

You specify an image file with Open Panel, load and display it in the view of the window. The size of the image will be 800x600 pixels. Depending on the size of the image, there may be vertical or horizontal margins to maintain the aspect ratio of the image.

Draw freehand lines

Draw a line on the view that is corresponded to the trajectory of mouse drag .

Erase lines

You create a rectangle area that contains you would erase , by dragging with right mouse button. Click Clear Button to erase the lines.

Output an image to a file

When you click Save Button, you can save a drawn image to a file of PNG format. The output destination file path is specified with Save Panel.

Implementation points

Draw a line

Draw a straight line between specified points. A line is drawn directly in Graphic Context with drawing methods of NSGraphicsContext class. As drawing a very short straight line each time a mouse drag event occurs, so it visually appears to draw a freehand line.

Erase lines

The erase operation erases all lines inside a specified rectangle. A line is a set of lines created by the occurrence of a mouse drag event. Each line has a different length depending on the speed of dragging. So if you erase a long line, it looks like the line outside the rectangle has been erased. To avoid this, the line on the boundary of the rectangle leaves the outer part of the rectangle.
